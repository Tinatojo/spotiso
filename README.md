# Spotiso  

A simple and effective way to download locally your Spotify musics.

## Installation  

1. **Clone the Repository:**  
   ```bash
   git clone <repository-url>
   cd spotiso

2. **Install Dependencies:**
   ```bash
   pipenv install

3. **Activate Virtual Environment:**
   ```bash
   pipenv shell


Run the Application:

    python main.py
